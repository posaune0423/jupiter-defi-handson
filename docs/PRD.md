​🤖 Agent Skillsとは？

​Agent Skillsとは、AIコーディングエージェント（Claude Code、Cursor、VS Codeなど）に特定のプロトコルを扱うためのコンテキストを与える、事前構築された指示セットです。単なるAPIドキュメントの読み込みとは異なり、手順レベルの実践的な知識を提供します。

​ビルダーが「JupiterのSwap APIを自分のアプリに統合して」と指示するだけで、エージェントは以下の要件を完全に理解した状態でコードを生成・実行します：

​/order + /execute の適切な使用

​x-api-key ヘッダーの付与

​スリッページやガスレスフローの処理・考慮

​💡 なぜハッカソンで重要なのか？ 通常なら統合に何時間もかかる作業を、JupiterのAgent Skillsを使えばAIアシスタントが自動で処理します。この圧倒的な時間短縮こそが、ハッカソンで勝つための強力な武器になります。

​🛠 ワークショップで重点的に扱うAPI

​1. Swap API

​👉 ハッカソンビルダーには、最もシンプルで全ルーター（RFQマーケットメーカー含む）にアクセスできる「ハッピーパス」を推奨します！

​ハッピーパス（推奨）： 全ルーターから最適価格で構築されたトランザクションを取得（/order & /execute）。署名して送信するだけで、トランザクションの着地はマネージドされます。

​上級者向け： カスタムトランザクション構築（CPI用の生のswap命令など、トランザクションを完全に制御可能）。

​2. Price API v3

​任意のSolanaトークンのUSD価格をmintアドレスから取得。

​最新のスワップ価格を基に算出。

​操作やウォッシュトレードを除外するためのヒューリスティックを活用。

​3. Tokens API

​トークン検索、メタデータ取得、検証ステータスの確認。

​オーガニックスコア（0〜100）： 実際の取引活動の健全性を評価可能。

​4. Trigger API v2

​USD価格ベースのトリガー注文、指値注文。

​利確 / 損切り（OCO）、条件付きチェーン（OTOCO）。

​注文はオフチェーンで保存され、デフォルトで非公開。

​5. Lend API

​デポジットボールトによる利回り獲得と担保を使った借入。

​Earn向けのREST APIが現在利用可能。

​6. Prediction Market API

​スポーツ、暗号資産、政治などの二択型予測市場データ。

​Polymarketとのパートナーシップによる提供。
